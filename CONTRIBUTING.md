# Contributing to PromptHub

GitHub 的贡献入口文件保留在仓库根目录，便于仓库访客和潜在贡献者快速找到协作说明。

详细指南请查看：

- [docs/contributing.md](./docs/contributing.md)
- [docs/README.md](./docs/README.md)
- [spec/README.md](./spec/README.md)

开始前建议先确认：

- 你要修改的是 `apps/desktop`、`apps/web` 还是 `packages/*`
- 非 trivial 改动先建立或更新 `spec/changes/active/<change-key>/`
- 提交 PR 前同步测试、用户文档与 `implementation.md`
