# stock-price 技能指南

## 适用场景
- 查询某只股票的最新价格、今日涨跌幅
- 获取历史K线数据（日线、分钟线等）
- 分析近期走势和成交量变化

## 前置条件
- MiniQMT 客户端已安装并保持登录运行状态
- xtquant 包已安装到 Python 环境中

## 可用脚本

| 脚本 | 功能 | 参数 |
|------|------|------|
| `scripts/get_kline.py` | 获取K线数据 | `<股票代码>` `[周期]` `[条数]` |

## 股票代码格式
- 沪市：600519.SH（贵州茅台）、688981.SH（中芯国际）
- 深市：000858.SZ（五粮液）、002594.SZ（比亚迪）、300750.SZ（宁德时代）

## 周期说明
- `1d` - 日K线（默认）
- `1m` / `5m` / `15m` / `30m` - 分钟K线
- `1h` - 60分钟K线

## 执行流程（必须遵守）
1. 识别股票名称，转换为带后缀的代码
2. 直接执行 `python skills/stock-price/scripts/get_kline.py <代码> [周期] [条数]`
3. 解读返回的 JSON 数据，给出最新价格、涨跌幅、走势分析

禁止：安装依赖、检查环境、验证包版本。

## 示例对话

用户: "茅台现在什么价？"
步骤:
1. 执行 `python skills/stock-price/scripts/get_kline.py 600519.SH 1d 5`
2. 读取最新一条数据的 close 价格
3. 返回最新价格和近期走势

用户: "中芯国际最近走势怎么样？"
步骤:
1. 执行 `python skills/stock-price/scripts/get_kline.py 688981.SH 1d 20`
2. 分析 20 天的收盘价走势、成交量变化
3. 返回走势分析

注意：所有路径参数使用相对路径（不带前导 /）。
