# codeview-skill

12当需要解释因子/指标的计算逻辑时，按以下顺序组织：

## 模板

### 1. 公式
```
[名称]：
公式本体
# 参数：符号 = 含义（单位）
```

### 2. 代入样例
具体数值代入，每步有数字，不跳步。

### 3. 计算展开
先中间结果，后最终结果。复杂时给运算链。

### 4. 取值参考（如有）
```
| 范围 | 含义 |
|---|---|
| > X | 解读 |
| X~Y | 解读 |
| < Y | 解读 |
```

### 5. 语义
- 因子方向（越大越好/越小越好）
- 取负号/对数后语义转换

## 示例（RSI_14）

```
RSI = 100 - 100 / (1 + RS)
# 参数：RS = avg_gain / avg_loss（相对强度），RSI ∈ [0, 100]，50=中性

代入：14日涨跌 +8,-3,+12,+5,-6,+9,-2,+15,-4,+7,-1,+11,-5,+3
avg_gain = 70/14 = 5.0
avg_loss = 21/14 = 1.5
RS = 5.0/1.5 ≈ 3.33
RSI = 100 - 100/(1+3.33) ≈ 76.9
RSI_14 = 76.9 - 50 ≈ +26.9  → 偏强，未超买

取值：>70超买 / 50~70偏强 / 50中性 / 30~50偏弱 / <30超卖
语义：偏离50越远，动能越强；+26.9表示近期上涨动能偏强
```

## 要点

- 样例数据要有具体数字，不能跳过步骤
- 边界情况要说清楚（RS=0 → RSI=0，RS→∞ → RSI=100）
