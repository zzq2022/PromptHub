# Definition of Done

在把任务标记为完成之前，至少确认以下事项：

## 文档

- 相关 `FR-*`、`DES-*`、`TEST-*`、`T-*` 已同步
- `README.md` 中的结构和命令说明仍然正确
- 如有关键技术取舍，已记录到 `docs/adr/`

## 代码与测试

- 高优先级需求有自动化验证
- 关键失败场景有覆盖
- 回归测试已更新
- Lint / build / test 可以稳定执行
- 白盒测试覆盖了关键分支和边界
- 与当前版本相关的性能和安全要求已验证，或明确写出延期原因

## 交付质量

- 不存在用默认值、空 catch 或假数据掩盖问题的实现
- 新增规则已沉淀到 `docs/rules/`，而不是只存在于临时沟通中
- 已知风险和未决事项已显式写出，而不是被隐藏
- 对关键疑点已经和用户完成确认，而不是开发者自行假定
