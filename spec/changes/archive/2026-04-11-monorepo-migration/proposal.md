# Proposal

Archived from historical planning content in `spec/changes/legacy/docs-08-todo/monorepo-migration.md`.

## Summary

Convert PromptHub from a single-package application into a pnpm monorepo that supports multiple product lines and shared packages.

## Why

- Enable shared packages across desktop and web
- Reduce duplication in types and database logic
- Create a scalable repository structure for future products

## Scope

### In Scope

- workspace split
- package extraction
- desktop relocation
- CI/CD updates

### Out Of Scope

- Full follow-through of every post-monorepo product initiative
