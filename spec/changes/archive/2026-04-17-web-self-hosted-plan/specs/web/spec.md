# Delta for Web Self-Hosted Plan

## ADDED Requirements

### Requirement: Browser-Accessible Self-Hosted Workspace

The system SHALL support a browser-accessible self-hosted web workspace with core PromptHub capabilities.

#### Scenario

- GIVEN a user deploys PromptHub Web
- WHEN the system is fully implemented
- THEN the user can manage prompts, folders, skills, settings, and media through the browser
