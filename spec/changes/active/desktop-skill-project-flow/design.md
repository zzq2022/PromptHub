# Design

## Approach

- 在 `SkillProjectsView` 内收口本次改动，避免为项目创建流程额外拆新组件。
- 将项目根目录字段前置，并在用户通过目录选择器选中路径后，根据目录名推导默认项目名。
- 添加项目成功后立即 `selectProject + scanProjectSkills`，减少一次显式刷新操作。
- 项目标识先采用首字母 avatar，不新增新的项目图标持久化字段；后续如需真实图标探测，再补充专门的文件系统能力。
- 将项目头部和项目列表头部统一为相近的高度、边框和内边距节奏，修复分割线错位。

## Tradeoffs

- 本轮不做项目真实图标探测，只提供稳定的首字母标识，优先解决当前缺少视觉锚点的问题。
- 自动填充项目名称仅在“名称为空”或“名称仍处于自动填充状态”时覆盖，避免误伤用户手动输入。
