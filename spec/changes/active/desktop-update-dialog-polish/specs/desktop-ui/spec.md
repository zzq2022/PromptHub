# Delta Spec

## Added

- 无。

## Modified

- 更新弹窗在发布说明较长、升级提醒可见时，必须保持在视口内显示，并通过弹窗内部滚动承载超长内容。
- 更新弹窗的升级备份提醒必须优先展示当前状态，而不是重复展示多段解释性文案。
- 更新弹窗中的备份确认勾选必须复用桌面端统一复选框视觉样式。

## Removed

- 更新弹窗中“未勾选确认时始终额外显示一条重复警告文案”的表现。

## Scenarios

- 当用户打开带长发布说明的更新弹窗时，弹窗本体不应超出窗口边界，超出内容应在弹窗内部滚动。
- 当用户下载完更新进入安装前确认时，备份区域应只保留标题、当前状态、操作按钮和勾选确认，不再堆叠重复说明。
