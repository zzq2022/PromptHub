# Delta Spec - 桌面端重构规范 (Desktop Refactoring Spec)

## Added

### 1. Codex 风格工作区 UI (Codex Workspace UI)
* **三栏式交互布局**：
  * **左栏（资源管理器树）**：折叠式展示当前项目工作区、我的提示词目录树（支持拖放移动）、我的 AI 技能、以及 IDE 规则模板文件。
  * **中栏（工作区编辑器）**：代码/提示词多标签页（Tabs）主编辑器，支持 Markdown 语法高亮与变量补全提示。
  * **右栏（大模型会话交互）**：集成的对话区域，用于即时预览和测试提示词的输出效果。
* **输入框底部配置状态栏 (Bottom Controls)**：
  * 新增**技能/MCP选择徽章栏**。用户点击“选择工具”，弹出悬浮菜单展示当前本地已启用的 Skills 和关联的 MCP Tools 列表。
  * 用户勾选后，选中的工具会以 Badge 形式贴在输入框下方。在大模型会话发送时，核心层会自动将这些 Skill 的 `content` 注入为系统提示词（System Prompt），并将 MCP 服务的方法列表转换为标准的 LLM Tool Call 声明。

### 2. 模型管理器 (Model Manager)
* **独立设置面板**：
  * 允许添加多个大模型提供商（OpenAI, Anthropic, DeepSeek, Google Gemini 等），配置 API 密钥（本地加密存储）、API 基础地址 (Base URL)、默认模型名称、以及最大上下文窗口。
  * 提供“测试连接”功能，即时发送单句 Ping 消息验证代理及网络通达性。

### 3. 自动同步后台服务 (Background Auto-Sync)
* **自动心跳同步**：
  * 引入基于定时器（如每 5 分钟）的周期性同步。
  * **防抖同步 (Debounced Sync)**：每当本地创建/更新/删除提示词、文件夹或技能时，启动一个 30 秒的防抖定时器，在用户停止操作后静默发起双向增量同步。
  * **状态指示器**：在 UI 右上角显示当前同步状态（`同步中`、`已是最新`、`同步冲突/断网`）。

### 4. 技能商店 (SkillHub Client)
* **公共技能发现页**：
  * 在桌面端独立菜单项中，支持直连并浏览 SkillHub 的云端公共技能。
  * 支持按分类（办公、开发、安全等）、标星数、周下载量进行检索和展示。
  * 单个技能支持展示 AI 自动安全审查打分（如 90分，安全等级 `safe`）。
  * 提供“一键安装”按钮，点击后自动从远程 `package_url` 下载 ZIP 依赖包，解包至本地 `~/.hermes/skills/`（或对应平台目录）并写入本地 SQLite 库。

## Modified

* **本地技能列表组件 (Skill List View)**：
  * 移除旧版简单的树状只读界面，将其嵌入 Codex 风格工作区的左栏中。
  * 为列表中的每个本地技能添加操作快捷键：`查看详情 (Readme)`、`安全扫描 (Scan)`、`发布申请 (Publish)`、`删除 (Delete)`。
* **技能同步与备份核心逻辑**：
  * 重构同步核心类 `SyncBackupCore`。原本只支持单向覆盖还原，现在在拉取远端快照时，先将其与本地数据按 `updatedAt` 时间戳进行字段级合并。本地较新的字段保持不变，远端较新的字段写入本地，最后再自动将本地最新的改动推送回远端，实现真双向合并。

## Scenarios

### 场景 1：用户修改本地提示词触发自动同步
1. 用户在 Codex 编辑器中修改了提示词 `A` 的 user_prompt，点击保存。
2. 本地数据库事务提交更新，触发主进程文件保存，并同时更新本地 `updated_at` 字段。
3. 自动同步模块捕获到数据库更改，启动 30 秒防抖计时。
4. 防抖结束，后台静默向服务端发起 `GET /api/sync/manifest` 校验哈希；发现服务端版本与本地不同，随后发起 `PUT /api/sync/data`，将本地修改安全同步至云端。右上角同步指示器由 `同步中...` 变为 `已是最新`。

### 场景 2：在输入窗口装配技能与 MCP 运行
1. 用户打开 Codex 交互会话，在底部输入框下方，点击 `+ 技能与工具` 按钮。
2. 弹出列表中展示了本地的 `Google Search Tool (MCP)` 和 `Git Diff Format (Skill)`。
3. 用户勾选了这两个工具，输入框下方随之出现两个带有“移除”按钮的 Badge 徽章。
4. 用户在输入框中输入：`帮我总结下最新项目提交`。
5. 点击发送，底层引擎自动将 `Git Diff Format` 技能的正文做为系统级 Prompt 拼接，同时将 `Google Search Tool` 对应的 MCP 方法注册在本次会话的大模型工具参数中。
