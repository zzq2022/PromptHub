# Desktop Delta

## Requirements

- Desktop settings must expose a preview-only entry for a future platform workbench UI.
- The preview entry must render without replacing the current Prompt or Skill workspace.
- The preview page must demonstrate a multi-resource management structure beyond Prompt and Skill.

## Scenarios

- User opens settings and sees a dedicated `Platform Preview` entry.
- User selects that entry and sees a platform workbench prototype page.
- User interacts with the prototype and sees list/detail context update locally.
