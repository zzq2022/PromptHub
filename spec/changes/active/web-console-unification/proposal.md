# 开发者控制台整合重提案 (V4 - Unified Console)

## 1. 背景与动因 (Background)
为了给用户提供最完整、高效的开发环境，同时避免功能割裂：
- 我们决定在网页端废弃自定义的卡片面板（Dashboard）。
- 将“我的技能”定位为**统一开发者工作台**，直接展现底层的 IDE 工作区。
- 用户在这个工作区下，既可以开发管理其“智能体技能 (Skills)”，又可以编辑其“提示词 (Prompts)”及编译规则。

---

## 2. 目标与范围 (Goal & Scope)
- **极简顶栏**：仅保留 `技能中心`、`我的技能` 和 `管理后台`。
- **无损整合**：在“我的技能”工作台下，保留完整的 IDE 功能（包括 Skills、Prompts、Rules、Projects）。
- **默认聚焦**：进入工作台后默认加载“Skills”面板，用户可通过左侧栏自由切换至“Prompts”。

---

## 3. 非目标与约束 (Non-Goal & Constraints)
- 桌面客户端（Desktop App）功能完全不受此变动影响。
